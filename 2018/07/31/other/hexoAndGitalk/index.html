hexo Next主题集成gitalk and hexo 集成next主题的个性化

在此之前你要搭建好了自己的博客，具体方法查看hexo文档就可以了。这里不做过多阐述

hexo Next主题集成gitalk

1、首先你要在GitHub上注册一个新的应用。注册位置在设置里面有一个developer setting，进去之后选择New OAuth App

点击之后会进入如下页面:

地址：https://github.com/settings/applications/new

参数说明：

  • Application name： 应用名称
  • Homepage URL：博客地址 （https://wsyWeb.github.io）
  • Application description # 描述
  • Authorization callback URL：博客地址，（https://wsyWeb.github.io）

注册之后会生成 Client ID 和 Client Secret， 这两个参数在后序会用到

2、在next主题下的/layout/_third-party/comments下新建一个gitalk.swig的文件夹（你也可以取别的名字），文件内容为：

{% if page.comments && theme.gitalk.enable %}
  <link rel="stylesheet" href="/css/gitalk.css">
  <script src="/js/src/gitalk.min.js"></script>

  <script type="text/javascript">
		var gitalk = new Gitalk({
		  clientID: '{{ theme.gitalk.ClientID }}',
		  clientSecret: '{{ theme.gitalk.ClientSecret }}',
		  repo: '{{ theme.gitalk.repo }}',
		  owner: '{{ theme.gitalk.githubID }}',
		  admin: ['{{ theme.gitalk.adminUser }}'],
		  id: location.pathname,
		  distractionFreeMode: '{{ theme.gitalk.distractionFreeMode }}'
		})
		gitalk.render('gitalk-container')
       </script>
{% endif %}

注意：这里我引入gitalk的css和js皆下载到了本地，线上引用地址为
css: https://unpkg.com/gitalk/dist/gitalk.css
js: /https://unpkg.com/gitalk/dist/gitalk.min.js
本地css为js放入文件夹为next主题下 source/css 、 source/js/src

创建好的gitalk文件引入layout/_third-party/comments/index.swig中，在index.swig中加入

{% include 'gitalk.swig' %}

修改next主题下/layout/_partials/comments.swig文件，找到该文件的最后一个if语句，加入代码

{% elseif theme.gitalk.enable %}
 <div id="gitalk-container"></div>

3、在next主题的_config.yml中加入以下内容

gitalk:
  enable: true
  githubID: 你的githubID 如：wsyWeb
  repo: 你的issue仓库地址，可以不是博客地址 如：wsyWeb.github.io
  ClientID: 刚刚申请的 ClientID
  ClientSecret: 刚刚申请的 ClientSecret
  adminUser: wsyWeb
  distractionFreeMode: true

ok，到这里你的评论功能就能出来了，登录认证后就可以正常使用了，如果你想修改评论的样式。在next下/source/css/_common/components/third-party中新建文件gitalk.styl。把你自定义的样式写在里面，在/source/css/_common/components/third-party/third-party.styl中引用样式就可以了

@import "gitalk";

关于hexo集成next主题的个性化设置

点击出现桃心特效

在/themes/next/source/js/src里面新建click-love.js文件，打开http://7u2ss1.com1.z0.glb.clouddn.com/love.js，将所有内容拷贝进去，在\themes\next\layout\_layout.swig文件末尾添加代码

<script type="text/javascript" src="/js/src/click-love.js"></script>

若不生效，hexo clean清除一下缓存再重新启动项目

个性化主题

next提供给用户自定义样式的style在文件themes\next\source\css\ _custom\custom.sty中，你可以在此引入图片，写入背景颜色等

更改footer

在/themes/next/layout/_partials/footer.swig中找到 if theme.footer.powered 这句话，这句以下都是footer的内容，你可以改成你想要的

hexo gitalk
